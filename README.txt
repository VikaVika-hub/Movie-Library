# Movie Library – Личная кинотека

Приложение с графическим интерфейсом для хранения, просмотра и фильтрации коллекции фильмов. Данные сохраняются в JSON-файл.

GitHub: [@VikaVika-hub] (https://github.com/VikaVika-hub)

## Описание

Пользователь может:
- добавить фильм с указанием названия, жанра, года выпуска и рейтинга (от 0 до 10)
- просматривать все добавленные фильмы в таблице
- фильтровать фильмы по жанру (выпадающий список) и по году выпуска
- сбрасывать фильтры
- данные автоматически сохраняются в `movies.json` и загружаются при запуске

## Установка и запуск

1. Убедитесь, что установлен Python 3.7+.
2. Скачайте файл `movie_library.py`.
3. Запустите:
   ```bash
   python movie_library.py
